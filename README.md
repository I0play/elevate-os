# Elevate OS

Elevate OS is a modern SaaS platform designed to help users improve focus, reduce stress, and enhance performance through daily guided sessions powered by AI.

## ✨ Features

- 🧠 AI-generated daily focus, calm, and performance sessions
- 🔄 Automated daily content updates (cron-based)
- 💳 Subscription system with Stripe
- 📊 User dashboard with session playback
- ☁️ Cloud-based storage with Supabase
- ⚡ Fast deployment with Vercel

## 🎯 What It Does

Elevate OS delivers short, guided mental conditioning sessions designed to help users:
- Enter deep focus states
- Reset stress and anxiety
- Improve performance and confidence
- Build consistent daily habits

## 🧱 Tech Stack

- Next.js (Frontend + API)
- Supabase (Database + Auth)
- Stripe (Payments)
- OpenAI (AI content generation)
- Vercel (Hosting + automation)

## 🚀 Getting Started

1. Clone the repository
2. Add environment variables
3. Deploy with Vercel
4. Set up Supabase + Stripe
5. Launch your SaaS

## 🔑 Environment Variables

OPENAI_API_KEY= SUPABASE_URL= SUPABASE_ANON_KEY= STRIPE_SECRET_KEY= STRIPE_PRICE_ID= BASE_URL=
